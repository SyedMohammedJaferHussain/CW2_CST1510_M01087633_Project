## Project Description
A streamlit app with multiple domains with secure authentication, data analysis, CRUD functions, and OpenAI ChatGPT API Implementation
## Features
- Secure password hashing using bcrypt with automatic salt generation
- User registration and login on streamlit
- Data Analysis
- CRUD Operations to create, read, update, and delete rows in tables
- AI Assistant in each dashboard to assist with any queries related to dashboard topic
## Technical Implementation
- Three dashboards each taking their own data and having their own classes and class objects
- SQLLite3 databases and tables for quick data entry and fetching
- Plotly, matplotlib, and streamlit charts for data analysis
- OOP Implementation for each table in SQLLite, Connection to SQLLite, Authorisation, and AI Assistant
